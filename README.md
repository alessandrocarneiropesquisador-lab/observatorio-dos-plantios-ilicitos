# observatorio-cannabis-mapa
Projeto de produção de um centro informacional georreferenciado de produção científica sobre o fenômeno dos plantios ilícitos no Brasil, sua erradicação e simbiose com processos de criminalização. Uma iniciativa do Nucleo de Estudos de Violência, Direitos Humanos e Políticas Públicas sobre Drogas - NEVIDH -UFJF
